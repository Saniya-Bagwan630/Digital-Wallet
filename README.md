# Digital Wallet System

A backend application built using Spring Boot.

## Features
- User creation
- Wallet management
- Add money
- Transfer money
- Transaction history

## Tech Stack
- Java
- Spring Boot
- JPA (Hibernate)
- H2 Database

## How to Run
1. Clone repo
2. Run Spring Boot application
3. Open H2 console at:
   http://localhost:8080/h2-console
